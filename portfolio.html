01.

Fetch My Treats

< webscraper in ruby >


I have an 11 year old rescue schnauzer called Sachi (short for Sachiko), and in my spare time, I foster rescue dogs.

One of my pet peeves (pun fully intended) is getting ripped off. Especially when it comes to Sachi's favourite treats, Greenies.

I built a web scraper in Ruby to compare the price of Greenies on various online pet stores, using open-uri and nokogiri.
02.

Clean My Home

< two sided marketplace using ruby on rails >


I'm a minimalist who enjoys clean code*...and a clean home. During my coding bootcamp, I built a two sided marketplace using Ruby on Rails, where vetted people can be paid to clean homes, and customers can book verified cleaners.

Includes location based search using Google maps, and a review system.

*This was built when I was first learning Ruby on Rails, so ironically the code is the one thing that is not particularly clean!

Make Me Popular

< instagram bot in python deployed on heroku >


When I was in Grade 10, one of my favourite teachers wrote in my school report: "Carmen has an appreciation for the absurd." That remains true to this day...and nothing is more absurd than human behaviour. I have deployed a python script that runs every hour (using Heroku Scheduler), creating a bot that interacts with people on Instagram. I'm constantly amused when people (and better yet, other bots) interact with my bot.

I'm currently working on running the script on a Raspberry Pi.

Measure my Metrics

< responsive mobile web app design with ui/ux & digital marketing optimisations >


Metrics. What software engineer doesn't love metrics?

I created a simple, mobile responsive landing page for a dance studio, Pole Athletica, and installed HotJar and Google Analytics to track user engagement and other relevant metrics. I have experimented with UI/UX A/B testing (wow that was a lot of letters), market surveys, and both SEO and SEM.
